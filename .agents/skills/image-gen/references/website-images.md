# Website Image Templates

Production-ready prompts for common website needs.

## Hero Banners

**Aspect Ratio**: 16:9 or 21:9 at 2K/4K

### Trades Services
```
A photorealistic wide shot of a professional [trade] at work
in a modern Australian home. Clean [colour] uniform with hi-vis vest,
confident posture, quality tools visible.
Natural light streaming through windows,
warm and inviting atmosphere.
Professional photography style, 16:9 aspect ratio.
```

### Professional Services
```
Modern office interior with floor-to-ceiling windows
overlooking [Australian city] CBD at golden hour.
Minimal furniture, warm timber accents,
single person working at standing desk.
Corporate but welcoming, architectural photography style.
16:9 aspect ratio, suitable for text overlay on right side.
```

### Hospitality
```
Inviting [cafe/restaurant] interior at golden hour,
natural timber and greenery, warm ambient lighting.
Tables set for service but no customers visible.
Australian contemporary design, food photography lighting.
16:9 aspect ratio.
```

## Service Cards

**Aspect Ratio**: 4:3 or 1:1 at 1K/2K

### Pattern
```
A [shot type] of [specific service action],
[key details that identify the service].
Professional setting, clean composition.
4:3 aspect ratio, suitable for card thumbnail.
```

### Examples

**Plumbing - Hot Water**:
```
Close-up of professional hot water system installation,
modern white unit mounted on wall,
copper pipes neatly connected, clear labels visible.
Clean, technical, professional. 4:3 aspect ratio.
```

**Electrical - Lighting**:
```
Modern kitchen with elegant pendant lights,
warm glow illuminating timber island bench.
Australian contemporary home interior.
4:3 aspect ratio.
```

## Infographics

**Aspect Ratio**: 1:1 or 4:3 at 2K

```
Create an infographic titled "[Title]"
showing [X] key points/steps:

1. [Item] - [icon description]
2. [Item] - [icon description]
3. [Item] - [icon description]

Style: Clean, modern, professional.
Primary colour: [hex code].
White background.
Clear visual hierarchy.
Large, legible text.
```

## Backgrounds

**Aspect Ratio**: 16:9 or 21:9

### Abstract
```
Abstract gradient mesh background,
soft teal and navy tones,
smooth colour transitions,
suitable for dark text overlay.
Minimal, professional, 21:9 ultrawide.
```

### Textured
```
Subtle concrete texture background,
light gray with fine grain,
minimal industrial aesthetic,
suitable for text overlay.
```

## Icons / Graphics

**Aspect Ratio**: 1:1

```
A simple [object] icon in flat vector style.
Single colour: [hex code].
Minimal detail, clean silhouette.
White background.
Suitable for 64x64 display.
```

Note: For true transparency, run through background removal after generation.

## Process Diagrams

```
Create a horizontal process diagram showing:

[Step 1] → [Step 2] → [Step 3] → [Step 4]

Each step has:
- Icon representing the action
- Short label below

Style: Flat design, connected by arrows.
Colours: [primary] for icons, [secondary] for arrows.
White background, clean typography.
```
