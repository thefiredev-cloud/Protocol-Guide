# Semantic Icon Mapping

Maps concepts to icons across Lucide, Heroicons, and Phosphor libraries.

## How to Use

1. Find the category matching your content
2. Look up the specific concept
3. Choose from recommended icons (first is best match)
4. Use the icon name for your chosen library

## Decision Tree

When the concept doesn't match tables above:

1. **Is it about achievement/quality?** → Trophy, Star, Award, BadgeCheck
2. **Is it about money/cost?** → Tag, DollarSign, Wallet, Percent
3. **Is it about place/location?** → MapPin, Globe, Map, Navigation
4. **Is it about people/users?** → User, Users, UserGroup
5. **Is it about communication?** → MessageCircle, Phone, Mail, Chat
6. **Is it about safety/security?** → Shield, Lock, ShieldCheck, Key
7. **Is it about time/speed?** → Zap, Clock, Timer, Calendar
8. **Is it industry-specific?** → Check Trade-Specific table
9. **Generic positive?** → CheckCircle, ThumbsUp, Sparkles
10. **Generic feature?** → Sparkles, Star, Zap
